# HeardU-Demo

**HeardU-Demo** is an AI-powered feedback collection agent designed to gather customer feedback about their restaurant visits.

This project demonstrates the core idea behind **HeardU**, the full platform that enables businesses to better understand customer experiences through automated conversations and feedback analysis.

The demo version focuses on the basic feedback collection workflow, allowing restaurants to interact with customers and capture their opinions about service, food quality, and overall experience.

The complete **HeardU** platform includes additional features such as advanced analytics, deeper customer insights, and enhanced feedback management tools.
